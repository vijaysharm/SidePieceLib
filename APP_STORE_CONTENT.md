## Description

SidePiece is a native macOS AI coding assistant that connects you to the world's most powerful language models — all from one polished, keyboard-driven interface.

**One app, every model.**
Stream responses from OpenAI, Anthropic, and OpenRouter through a single unified interface. Switch between GPT, Claude, DeepSeek, Gemini, Llama, and hundreds of other models without juggling browser tabs or separate apps. Bring your own API keys and stay in control of your costs.

**Built for your codebase.**
Open any project folder and SidePiece indexes it instantly. Ask questions about your code, search across files, read source with line numbers, run regex queries, and navigate directory trees — all without leaving the conversation. Six built-in codebase tools give the AI deep, read-only access to your project so answers are grounded in your actual code, not guesses.

**Real-time streaming that feels instant.**
Responses stream token-by-token as the model thinks. Extended thinking and reasoning traces are displayed inline so you can follow the AI's chain of thought in real time. Streaming is fast, cancellable, and works reliably across every supported provider.

**Conversations that persist.**
Every conversation is automatically saved to disk. Pick up where you left off, browse past sessions, or start fresh — your history is always there when you need it. Conversations are organized per-project so context never bleeds across codebases.

**Context you control.**
Attach files, folders, and tools to any conversation using an @-mention interface. Drag and drop images directly into the input field. The context overlay gives you fine-grained control over exactly what the AI can see, keeping responses focused and relevant.

**Secure by design.**
API keys are stored in the macOS Keychain — never in plain-text config files. Your code stays on your machine; SidePiece connects directly to provider APIs with no intermediary servers.

**A native Mac experience.**
SidePiece is built entirely in SwiftUI. It supports keyboard shortcuts, multi-project navigation, model selection, inline attachments, and a responsive sidebar — all designed to feel like a first-class Mac app, not a wrapped web view.

**Open source and transparent.**
SidePiece is fully open source. Inspect every line of code, contribute improvements, or build your own AI assistant on top of the library. No black boxes, no hidden telemetry — just honest software you can trust and verify.

Whether you're debugging a tricky issue, exploring an unfamiliar codebase, drafting documentation, or just need a second opinion on an architecture decision, SidePiece puts the best AI models at your fingertips without leaving your workflow.
